# EXP28：示波器实验（虚拟仿真实验）

| 学时 | 实验难度 | 实验报告难度 |
|------|---------|------------|
| 2 | ⭐ | ⭐ |

## 实验内容

  虚拟仿真实验。全程在电脑上完成。

  通过实验需要在模拟实验中取得 80 分以上的总成绩。

  如果掌握标准化流程的话，可以在 10 分钟内达到90+。在仿真模拟平台上有实验指导。

  大部分工作可以在宿舍完成。**强烈建议先在宿舍完成。**

  你仍然需要线下前往指定教室获得实验报告的签名，但你只需要展示你在宿舍获得的李萨如图形就可以结束。

  实验难度较低而不甚耗时 ~~(除非你没有按照建议流程做)~~，非常推荐选择。

## 建议流程

  1. 在自己电脑的Windows上安装好环境（.exe文件，注意要关闭防火墙和其他杀毒软件）；
  2. 按照实验指导，先后完成“学习模式”和“实验模式”，记录数据，将三个李萨如图形截图（详见下文“注意事项”）；
  3. 实验课当天直接把截图的李萨如图形给老师看，拿到教师签名。

- **非常不建议**的流程：在课上先做“实验模式”，回到宿舍再做“学习模式”。具体原因详见“补充”。
    
## 实验报告

  报告所需要测定的数据都可以直接在电脑上读数。

  值得注意的是，由于所有同学都使用同一套仿真模拟系统，因此数据不会有什么变化。

  唯一涉及随机的是待测信号源的频率，每次实验的待测信号源频率都是随机值，所以请特别留意这一点。

## 注意事项

以下注意事项在实验指导中没有提及，但了解它们可以帮助你快速获得高分：
  - 如有必要，请使用更优秀的 AI 辅助你进行作业。有条件的可以使用 Gemini，没有条件的请使用 Kimi，GLM 等。特别地，如果你实在要使用豆包，请使用思考/专家模式。无论如何不要使用快速模式。
  - 在学习模式中，你必须**每完成一次操作就点击一次“记录数据”按钮并保存相应的内容**，否则你的学习模式只能得到2.5分而无法通过实验。完成一个操作后，可以**双击右下角**切换到下一个操作。
  - 记得截图对应的李萨如图形，以便在线下环节向对应的老师进行展示。
  - 可以通过待测信号源接入 CH2 并关闭 CH1，在示波器的屏幕上读取其频率。这样可以直接算出信号发生器需要设定的频率，而不需进行大量的尝试。
  - 在做完全部实验前，任何情况下，
    - **<mark>不要点击“结束操作”！！！</mark>**
    - **<mark>不要点击“结束操作”！！！</mark>**
    - **<mark>不要点击“结束操作”！！！</mark>**
    
    你应该做的是**点击屏幕最上方的 HOME 键回到主页**，开始“实验模式”的操作，**否则你将不得不重新完成所有的实验步骤。**
  - 在实验模式中，你不需要完成所有学习模式中的操作。你需要完成的有：
    - 校准示波器（同学习模式的操作）
    - 调出 3 个李萨如图形（不需要截图）
    - 当然，还有填充“记录数据”中的表格。
  - 由于除李萨如图形之外的所有数据是确定而非随机的，你可以参考本仓库下的其它实验报告来检验自己所读出的数据是否正确。

如有缺漏，欢迎同学们进行补充。祝大家一次通关。

## 补充


- 刷分：对“学习模式”下的某些选择题没有把握时，可以：
    1. 查看当前分数
    2. 更改选项
    3. 点击“记录数据”按钮并保存相应的内容
    4. 刷新“成绩查看”页面，观察分数变化

- 经测试，**将整个实验系统关闭** or **点击“结束操作”**，成绩会直接上传，不会保留进度。

    也就是说，如果你像我一样没有在宿舍提前完成，选择**在课上先做“实验模式”，回到宿舍再做“学习模式”**，那么你很可能会发现你的实验成绩只有三四十分。

    这是因为你实际上获得了两次成绩：一次是课上做的“实验模式”成绩，“学习模式”成绩是0分；一次是回宿舍做的“学习模式”的成绩，“实验模式”成绩是0分。

    所以你回到宿舍，还得从头一次性把两个模式都做一遍，才能获得一个完整的成绩，非常浪费时间。

    不确定是否存在课上做一部分、回宿舍做另一部分，可以获得正常成绩的案例，欢迎补充。

    本人不仅没有提前做，还不小心点到了“结束操作”，被硬控时长高于两小时，望周知。T^T
