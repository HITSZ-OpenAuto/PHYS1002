## 实验描述

实验过程不算困难，基本上都能按时完成。

- 老师要求在课上就完成计算转换系数 $K$，以及后面几个测量值的平均数，所以建议带上一个科学计算器。
  $K$ 的理想值是大约 $9\times 10^{-3} N/mV$，注意数量级和单位换算。
- 如果你在做实验的时候发现水加热得很慢或者一直加热了但水温不变，可能是因为温度探头没有被水完全浸没，这可以通过降低物块升高液面解决。
