## 实验描述
实验操作简单，是原理最简单的实验之一，槽点在于实验器材以及油滴选取，实验器材有按键不灵的情况，很可能导致油滴掉出框内、时间测量不准确的问题。老师有一定数据要求，个人建议通过自救群的速成表来快速计算误差调整数据减小误差。
