## 实验描述

本实验主要难在读数：
- 本实验底座标尺实际是由两个 60cm 拼接而成，中间不连续，一开始没注意容易误读
- 器材上并没有明显标识（如箭头）辅助读数，所得数据误差会非常大，老师检查数据时也不容易找出错误
- 由于需要读的数太多，加上实验过程中会熄灯，很容易让人眼花而读得错误结果
